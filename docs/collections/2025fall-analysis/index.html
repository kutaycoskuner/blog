Unit 1: Set Theory, Mappings, and Logical Conclusions

1.1. Basics of Sets (Grundlegendes über Mengen)

This section introduces the foundational concepts and operations of set theory.

Core Concepts

Set (Menge): A collection of distinct objects, called elements.

  • Notation: M={x∣P(x) is true}\quad M = \{x \mid P(x) \text{ is true}\} M={x∣P(x) is true}

Element: An object belonging to a set.

  • Notation: x∈M(x is an element of M)\quad x \in M \quad (\text{x is an element of } M)x∈M(x is an element of M)

Equality of Sets (Gleichheit von Mengen): AAA and BBB have exactly the same elements.

  • Notation: A=B\quad A = BA=B

Subset (Teilmenge): All elements of AAA also belong to BBB.

  • Notation: A⊂B\quad A \subset BA⊂B

Empty Set (Leere Menge): The set containing no elements.

  • Notation: ∅\quad \emptyset∅

Set Operations (Mengenoperationen)

Intersection (Schnittmenge): Elements belonging to both AAA and BBB.

  • Notation: A∩B={ x:x∈A  ∧  x∈B }\quad A \cap B = \{\, x : x \in A \;\wedge\; x \in B \,\}A∩B={x:x∈A∧x∈B}

Union (Vereinigung): Elements belonging to AAA, or BBB, or both.

  • Notation: A∪B={ x:x∈A  ∨  x∈B }\quad A \cup B = \{\, x : x \in A \;\vee\; x \in B \,\}A∪B={x:x∈A∨x∈B}

Set difference (Differenzmenge): Elements in AAA but not in BBB.

  • Notation: A∖B={ x:x∈A and x∉B }\quad A \setminus B = \{\, x : x \in A \text{ and } x \notin B \,\}A∖B={x:x∈A and x∈/B}

Disjoint Sets (Disjunkte Mengen): AAA and BBB have no common elements.

  • Notation: A∩B=∅\quad A \cap B = \emptysetA∩B=∅

Ordered Pair (Geordnetes Paar): A pair of objects where the order matters. (such as coordinates)

  • Notation: (a,b)≠(b,a)(if a≠b)\quad (a, b) \neq (b, a) \quad (\text{if } a \neq b)(a,b)=(b,a)(if a=b)

Cartesian Product (Kartesisches Produkt / Produktmenge): The set of all possible ordered pairs (a,b)(a,b)(a,b).

  • Notation: A×B={ (a,b):a∈A and b∈B }\quad A \times B = \{\, (a,b) : a \in A \text{ and } b \in B \,\}A×B={(a,b):a∈A and b∈B}

1.2. Basics of Mappings/Functions (Grundlegendes über Abbildungen)

Definition 1.12 (Mapping/Function - Abbildung, Funktion)

A mapping fff maps a set XXX to a set YYY.

  • Notation: f:X→Y,f: X \rightarrow Y,f:X→Y, x↦f(x)x \mapsto f(x)x↦f(x). The element x∈Xx \in Xx∈X is assigned to the element f(x)∈Yf(x) \in Yf(x)∈Y.
  • Domain (Definitionsbereich): dmn(f):=Xdmn(f) := Xdmn(f):=X.
  • Image (Bild): The set of all elements in YYY that are reached by fff: im(f):={y∈Y:∃x∈X with y=f(x)}im(f) := \{y \in Y: \exists x \in X \text{ with } y = f(x)\}im(f):={y∈Y:∃x∈X with y=f(x)}
  • Graph (Graph): The set of all ordered pairs: F={(x,f(x)):x∈X}F = \{(x, f(x)): x \in X\}F={(x,f(x)):x∈X}
  • If the target set YYY is R\mathbb{R}R, C\mathbb{C}C, or Rn\mathbb{R}^{n}Rn, the mapping is often called a Function (Funktion).

Definition 1.15 (Image and Preimage - Bild und Urbild)

For a mapping f:X→Yf: X \rightarrow Yf:X→Y and subsets X′⊂XX' \subset XX′⊂X, Y′⊂YY' \subset YY′⊂Y:

  • f-Image of X′X'X′ (f-Bild von X′X'X′): f(X′)={f(x):x∈X′}⊂Yf(X') = \{f(x): x \in X'\} \subset Yf(X′)={f(x):x∈X′}⊂Y
  • f-Preimage of Y′Y'Y′ (f-Urbild von Y′Y'Y′): f−1(Y′)={x∈X:f(x)∈Y′}f^{-1}(Y') = \{x \in X: f(x) \in Y'\}f−1(Y′)={x∈X:f(x)∈Y′}

Definition 1.17 (Composition - Verkettung)

Given mappings f:X→Yf: X \rightarrow Yf:X→Y and g:Y→Zg: Y \rightarrow Zg:Y→Z, the composition (Verkettung) of ggg and fff is defined as:

  • g∘f:X→Zg \circ f: X \rightarrow Zg∘f:X→Z
  • (g∘f)(x)=g(f(x))(g \circ f)(x) = g(f(x))(g∘f)(x)=g(f(x)) for x∈Xx \in Xx∈X

1.3. Basics of Logical Conclusions (Grundlegendes über logische Schlussfolgerungen)

Mathematics derives statements from fundamental Axioms (Axiome) using formal logic.

Definition 1.18 (Implication - Implikation)

  • A⇒BA \Rightarrow BA⇒B: The correctness of statement AAA implies that statement BBB is also correct.
  • For conditions P(x)P(x)P(x) and Q(x)Q(x)Q(x), P(x)⇒Q(x)P(x) \Rightarrow Q(x)P(x)⇒Q(x) means that Q(x)Q(x)Q(x) is true for an element whenever P(x)P(x)P(x) is true.

Lemma 1.21 (Set Interpretation) Let A:={x:P(x) is true}A:=\{x: P(x) \text{ is true}\}A:={x:P(x) is true} and B:={x:Q(x) is true}B:=\{x: Q(x) \text{ is true}\}B:={x:Q(x) is true}.

  • The implication P(x)⇒Q(x)P(x) \Rightarrow Q(x)P(x)⇒Q(x) is equivalent to the subset relation A⊂BA \subset BA⊂B.

Definition 1.22 (Equivalence - Äquivalenz) Two statements AAA and BBB are called equivalent (äquivalent) if both A⇒BA \Rightarrow BA⇒B and B⇒AB \Rightarrow AB⇒A hold.

  • Notation: A⇔BA \Leftrightarrow BA⇔B.
  • Read as: A holds if and only if (genau dann, wenn - gdw.) B holds.

Unit 2: Real and Complex Numbers (Reelle und komplexe Zahlen)

2.1. Real Numbers (Reelle Zahlen)

The set of Real Numbers (R\mathbb{R}R) is a set with two operations (addition +++ and multiplication ⋅\cdot⋅) and an order relation (<<<) that satisfy 13 axioms.

2.1.1. Field Axioms (Körperaxiome)

These 9 axioms govern the arithmetic operations in R\mathbb{R}R. We use the quantifiers: ∀\forall∀ (for all / für alle) and ∃\exists∃ (there exists / es existiert).

AxiomAdditionMultiplicationLaw (Gesetz)

2.1.2. Order Axioms (Anordnungsaxiome)

These axioms define the set P⊂R\mathbb{P} \subset \mathbb{R}P⊂R of positive numbers (positive Zahlen):

  1. (Trichotomy) ∀x∈R\forall x \in \mathbb{R}∀x∈R exactly one of the following holds: x∈Px \in \mathbb{P}x∈P, x=0x=0x=0, or −x∈P-x \in \mathbb{P}−x∈P.
  2. (Closure under Addition) If x∈P∧y∈Px \in \mathbb{P} \land y \in \mathbb{P}x∈P∧y∈P, then x+y∈Px+y \in \mathbb{P}x+y∈P.
  3. (Closure under Multiplication) If x∈P∧y∈Px \in \mathbb{P} \land y \in \mathbb{P}x∈P∧y∈P, then xy∈Pxy \in \mathbb{P}xy∈P.

Definition 2.8 (Inequalities - Ungleichungen)

  • xxx is positive (positiv): 0<x0 < x0<x, if x∈Px \in \mathbb{P}x∈P.
  • xxx is negative (negativ): 0>x0 > x0>x, if −x∈P-x \in \mathbb{P}−x∈P.
  • xxx is smaller than yyy (kleiner als yyy): x<yx < yx<y, if 0<y−x0 < y-x0<y−x.

Lemma 2.10 (Rules for Inequalities) For arbitrary x,y,u,v,z∈Rx, y, u, v, z \in \mathbb{R}x,y,u,v,z∈R:

  • Transitivity (Transitivität): (x<y∧y<z)⇒x<z(x < y \land y < z) \Rightarrow x < z(x<y∧y<z)⇒x<z.
  • Translation Invariance (Translationsinvarianz): x<y⇔x+z<y+zx < y \Leftrightarrow x+z < y+zx<y⇔x+z<y+z.
  • Multiplication Rule: If x<yx < yx<y:
    • 0<z⇒zx<zy0 < z \Rightarrow zx < zy0<z⇒zx<zy.
    • 0>z⇒zx>zy0 > z \Rightarrow zx > zy0>z⇒zx>zy.

2.1.3. Completeness Axiom (Vollständigkeitsaxiom)

Definition 2.15 (Upper Bound - Obere Schranke)

  • A set M⊂RM \subset \mathbb{R}M⊂R is bounded above (nach oben beschränkt) if ∃s∈R\exists s \in \mathbb{R}∃s∈R such that sss is an upper bound (obere Schranke) of MMM (i.e., ∀x∈M:x≤s\forall x \in M: x \le s∀x∈M:x≤s).

Definition 2.17 (Supremum/Least Upper Bound - Supremum/Kleinste obere Schranke)

  • s∈Rs \in \mathbb{R}s∈R is the supremum (or least upper bound) of MMM (Notation: s=sup⁡Ms = \sup Ms=supM) if:
    1. sss is an upper bound of MMM.
    2. ∀ϵ>0∃x∈M:x>s−ϵ\forall \epsilon > 0 \exists x \in M: x > s - \epsilon∀ϵ>0∃x∈M:x>s−ϵ (i.e., no number smaller than sss is an upper bound).

Axiom 2.19 (Completeness Axiom - Vollständigkeitsaxiom)

  • Every non-empty set M⊂RM \subset \mathbb{R}M⊂R that is bounded above (nach oben beschränkt) has a supremum (Supremum) s=sup⁡Ms = \sup Ms=supM in R\mathbb{R}R.

2.2. Complex Numbers (Komplexe Zahlen)

The set of Complex Numbers (C\mathbb{C}C) extends R\mathbb{R}R by introducing the imaginary unit (imaginäre Einheit) iii with i2=−1i^2 = -1i2=−1. A complex number zzz is written as z=x+iyz = x + iyz=x+iy, where xxx is the real part (Realteil) Re(z)Re(z)Re(z) and yyy is the imaginary part (Imaginärteil) Im(z)Im(z)Im(z), with x,y∈Rx, y \in \mathbb{R}x,y∈R.

Unit 3: Natural Numbers and the Principle of Induction (Natürliche Zahlen und Induktionsprinzip)

3.1. The Natural Numbers (Die natürlichen Zahlen)

Definition 3.1 (Inductive Set - Induktive Menge)

  • A set M⊂RM \subset \mathbb{R}M⊂R is called inductive (induktiv) if:
    1. 1∈M1 \in M1∈M
    2. ∀n∈R:n∈M⇒n+1∈M\forall n \in \mathbb{R}: n \in M \Rightarrow n+1 \in M∀n∈R:n∈M⇒n+1∈M.

Definition 3.3 (Natural Numbers - Natürliche Zahlen)

  • The set of Natural Numbers (N\mathbb{N}N) is defined as the unique inductive set that is a subset of all other inductive sets: N:=⋂{M⊂R:M is inductive}\mathbb{N} := \bigcap \{M \subset \mathbb{R}: M \text{ is inductive}\}N:=⋂{M⊂R:M is inductive}

3.2. Principle of Induction (Das Induktionsprinzip)

Satz 3.5 (Principle of Mathematical Induction - Das Prinzip der vollständigen Induktion) Let A(n)A(n)A(n) be a statement depending on n∈Nn \in \mathbb{N}n∈N. If the following two conditions are met:

  1. Base Case (Induktionsanfang - IA): A(1)A(1)A(1) is true.
  2. Inductive Step (Induktionsschritt - IS): For all n∈Nn \in \mathbb{N}n∈N, the implication A(n)⇒A(n+1)A(n) \Rightarrow A(n+1)A(n)⇒A(n+1) is true.
  • Then the statement A(n)A(n)A(n) is true for all n∈Nn \in \mathbb{N}n∈N.
