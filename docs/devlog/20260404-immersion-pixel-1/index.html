Devlog: Custom Shader Grass Meadow

I am creating a template for utilizing custom shaders in web for more artistic control, I will attempt to replicate t3ssel8r’s environmental 3d pixel art. The first step was building a simple grass meadow prototype.

  1. Created plane geometry for ground and 2D planar grass blades.

  2. Generated some points on geometry to place grass patch representing quads.

  3. As number of grass patches will be large to convince a meadow, I needed an optimized solution. Used instancedMesh class of three.js where we delegate computational heavy work to GPU to avoid CPU bottlenecks. Encountered a problem that, custom shader does not accept instancedMatrix so I pass them to my custom shaders as attribute.

  4. When we move around scene, grass blades are looking the same direction as this breaks illusion. I used technique called billboarding, basically orienting directions of grass blades to view direction. However, restricted it only on yaw (x-axis rotation) for now.

  5. Instead of quads, created a basic 128 pixel image for grass patch and pass to my shaders to render this texture on quads.

This is the final result at this moment. As it is not a work-in-progress version I am unable to provide interactive version. Feedback and suggestions are very welcome.
